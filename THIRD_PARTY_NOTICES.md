## ProgrammerArt bundled textures

Ledger bundles the **ProgrammerArt** texture pack for built-in block textures.

- Source project: https://github.com/deathcap/ProgrammerArt
- Author/project: deathcap / ProgrammerArt
- License: CC BY 4.0

The packaged copy and upstream license/readme are included under:

```text
vendor/programmerart/
```

These bundled textures are used so Ledger can ship with built-in block textures without redistributing Mojang's default Minecraft textures.
